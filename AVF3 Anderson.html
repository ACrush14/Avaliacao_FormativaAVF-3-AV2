Deseja-se fazer uma pesquisa em um grupo de atletas, para os quais foi solicitado a altura e o sexo (M ou F) dessas pessoas.
Faça um programa em linguagem JavaScript que calcule e escreva:
>A média de altura das mulheres.
>Número de homens
O programa deve encerrar a solicitação de informações quando ler uma altura igual a zero.
Não devem ser usados vetores ou funções matemáticas prontas.
O arquivo deve ser enviado no formato HTML
Não perguntar total de pessoas.
Parar quando ler uma altura igual a zero
